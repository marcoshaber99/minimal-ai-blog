## Getting Started

### Prerequisites

- Node.js (v18 or later)
- PostgreSQL database

### Installation

1. Clone the repository:
2. Install dependencies:

```shellscript
npm install
```

3. Set up your environment variables:

```shellscript
cp .env.example .env
```

4. Set up your database:

```shellscript
npx prisma db push
```

5. Run the development server:

```shellscript
npm run dev
```
